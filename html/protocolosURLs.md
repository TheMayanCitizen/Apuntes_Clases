### Protocolos en la URL del encalce

```html
<!-- El protocolo mailto es un protocolo de correo electrónico. -->
<a href="mailto:userexample@dominio.com?subject=Hazme una cotizacion"
  >Enviar email a userexample@dominio.com</a
>

<!-- Whatsapp link protocol -->
<a href="https://wa.me/+12345678?text=tu%20Mensaje%20De%20Texto" target="_blank"
  >Enviar mensaje a Whatsapp</a
>
<a href="whatsapp://send?phone=+123456789" target="_blank"
  >Enviar mensaje a la app de whatsapp</a
>

<!-- Tel link protocol -->
<a href="tel:+123456789">Click para llamar</a>

<!-- Skype link protocol  -->
<a href="skype:usuario_skype?call">Llamar a usuario_skype</a>

<!-- zoommtg link protocol -->
<a href="zoommtg://zoom.us/join?confno=123434566"
  >Unirme a la reunión de Zoom</a
>

<!-- slack -->
<a href="https://espaciodetrabajo.slack.com/archives/canalid"
  >Únete a nuestro espacio en slack</a
>

<!-- Discord -->
<a href="https://discord.gg//codigodeinvitacion"
  >únete a nuestro servidor en Discord</a
>
```
